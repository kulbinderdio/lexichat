# LexiChat Feature Video — Caption Script

**Target length:** 3–4 minutes  
**Format:** Screen recording with on-screen caption overlays. No voiceover.  
**Caption style:** Large, clean sans-serif (e.g. Inter or SF Pro). White text, semi-transparent dark pill background. Centred bottom of screen unless noted.  
**Music:** Optional soft ambient track at low volume.  
**Pacing:** Each caption stays on screen for 3–4 seconds. Action happens first, caption appears 0.5s after.

## Caption Design Spec

| Element | Value |
|---------|-------|
| Font | Inter Bold (or SF Pro Display) |
| Main caption size | 28–32px at 1080p |
| Subtitle/detail size | 20–22px at 1080p |
| Text colour | White `#ffffff` |
| Background | `rgba(0, 0, 0, 0.55)` rounded pill |
| Padding | 10px 20px |
| Position | Centred, 80px from bottom |
| Animation | Fade in 0.2s, hold, fade out 0.2s |
| Accent captions | Indigo `#818cf8` text, no background pill |

## Production Notes

- **Highlight key UI elements** with a subtle zoom or a brief circle/arrow animation when a caption references something specific on screen (e.g. the ⚙ indicator, the paperclip, the profile switcher)
- **Pause the screen action** for 0.5s before each caption appears — let the viewer see the result first, then explain it
- **Tool indicator moments** (⚙ web_search, ⚙ read_file, ⚙ write_file) are the strongest visual hooks — slow down slightly here
- **Outro fade** should be 1–2 seconds to black before the logo appears
- **Recommended tool:** ScreenStudio (macOS), Rotato, or DaVinci Resolve for caption overlays
