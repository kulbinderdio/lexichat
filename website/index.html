Persistent Long-Term Memory

Most assistants forget everything the moment you close the chat. LexiChat keeps a personal wiki — its long-term memory. Tell it something worth keeping and it files the fact away on its own; ask about it in a brand-new conversation and it searches the wiki, reads the page back, and picks up right where you left off. Perfect for todo lists, project notes, people, dates, and preferences that should outlive a single chat.

The wiki is just plain Markdown files on your own disk — no database, no cloud. Open them in any editor, edit them by hand, back them up, or delete them. They're yours.

Python Code Sandbox

Let Lexi write and run Python to compute exact answers — calculations, data processing, parsing and logic — instead of estimating. Every run happens in an isolated interpreter with no network and no surprise file access. You approve execution before it starts, and code can only touch the folders in your sandbox.

Scheduled Jobs & Workflows

Define prompts that run automatically — daily, weekly, every N hours, or on demand. Each job is tied to a specific profile and its tools, so it always runs with the right context regardless of which profile you have active.

Use the Step Builder to design multi-step workflows: chain text instructions with specific tool calls (web search, file read, datetime, and more) into a precise, repeatable sequence your agent follows every time.

Getting Started

Up and running in minutes.

Three steps. No accounts. No API keys.

1

Install Ollama

Download Ollama for your platform and pull a model. ollama pull gemma3 is a great starting point — it's fast and capable.

2

Download & install LexiChat

Download the installer for your platform from the link above. macOS (Universal), Windows, and Linux (AppImage + .deb) all supported.

3

Start chatting

Open LexiChat, select your model, and start a conversation. Add folders to the Sandbox to give Lexi access to your files, or connect external APIs via the Tools panel.
